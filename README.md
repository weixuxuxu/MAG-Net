# MAG-Net
This repository contains the code for 'MAG-Net: A Multi-Task Deep Learning Framework for Thymic Tumor Diagnosis'.

The code is written in Python and requires the following packages:
- Python 3.8.15
- PyTorch 1.12.1
- Torchvision 0.13.1
- Tensorboard 2.12.3
- Monai 1.2
- Scikit-learn 1.2.0
